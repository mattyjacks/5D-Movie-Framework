# Video Sections for 5D Movie Interface

This folder contains video sections for the Detroit Interactive Universe 5D movie experience.

## Folder Structure

Place your video files in the following structure:

```
public/videos/
├── main/                    # Main timeline videos
│   ├── section-1.mp4
│   ├── section-2.mp4
│   └── section-3.mp4
├── reality-a/               # Alternate Reality A (Marvel-like timeline)
│   ├── section-1.mp4
│   ├── section-2.mp4
│   └── section-3.mp4
├── reality-b/               # Alternate Reality B (DC Comics intersection)
│   ├── section-1.mp4
│   ├── section-2.mp4
│   └── section-3.mp4
├── memories/                # Memory flashback sequences
│   ├── sam-raimi-detroit.mp4
│   ├── crimewave-set.mp4
│   ├── jason-bourne-moment.mp4
│   └── awakening.mp4
└── tech/                    # Technology/Sentient cars sequences
    ├── avatar-tech.mp4
    ├── sentient-cars.mp4
    └── time-travel.mp4
```

## Video Requirements

- **Format**: MP4 (H.264 codec recommended)
- **Resolution**: 1920x1080 or higher
- **Aspect Ratio**: 16:9
- **Duration**: 10-30 seconds per section for optimal 5D navigation

## 5D Dimensions

1. **Time** - Linear progression through the story
2. **Space** - Multiple camera angles and perspectives
3. **Reality** - Parallel universes and alternate timelines
4. **Memory** - Flashbacks and character recollections
5. **Choice** - User-driven narrative branches

## Getting Started

For the initial demo, you can use placeholder videos or stock footage that matches the Detroit/espionage/superhero theme. The interface will automatically detect and display available video sections.

## Detroit Interactive Universe (DIU) Scenes

Recommended scenes to film/create:
- Detroit cityscapes (establishing shots)
- Sam Raimi homage sequences
- Covert operation sequences (Jason Bourne-style)
- Futuristic car technology demonstrations
- Time travel/Avatar technology effects
- Memory awakening sequences
